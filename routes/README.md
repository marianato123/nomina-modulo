# 💰 Módulo de Nómina — CONEXVA ERP

Módulo independiente de nómina desarrollado para integrarse al ERP CONEXVA de Moreno Rodríguez y Asociados.

## ¿Qué hace este módulo?

- Lista todos los empleados del despacho
- Calcula automáticamente IMSS, ISR e INFONAVIT
- Genera recibos de nómina por empleado
- Interfaz visual con el estilo de CONEXVA

## Tecnologías

- Node.js + Express (backend)
- HTML + CSS (frontend)
